# pubbr
This is a repository of sample JavaScript and JS-containing Epub files to support the development and testing of Pubbr, an Epub file reader app for
Android. 

Epub files contain Xhtml, not Html, files. The former are subject to stricter compliance criteria. Thus, JavaScript code that works in Html might not work in Xhtml.

Pubbr was developed specifically with Epub JavaScript support in mind. An example would be dynamic colours. For this reason, the app will not feature a dark theme.
